環境構築

開発環境

docker composeを使ってデバッグできるようにしているので，基本的にそれを使ってください

Gitのインストール

Git - Downloadsからインストール
(参考)WindowsにGitをインストールする手順(2022年9月更新)

Gitのパスが通っていることを確認

$ git -v
git version 2.37.3

ユーザ名とメールアドレスの登録

git config --global user.name "Your name"
git config --global user.email "email@example.com"

Docker desktopのインストール

docker-desktopからインストール

使用するOSに合ったものをダウンロードしてください

インストーラーに従いパスが通っていることを確認

$ docker --version
Docker version 20.10.17, build 100c701

$ docker compose version
Docker Compose version v2.10.2

ホームページのデバッグ方法

リポジトリのクローン

git clone https://github.com/Grandelfino/homepage.git
cd homepage/

.envファイルの作成

touch .env

.envファイルにGrandelfino/secretsのContentful ENVの内容をコピーする

CONTENTFUL_SPACE_ID=xxx
CONTENTFUL_ACCESS_TOKEN=xxx

localhostでのホスティング
homepageディレクトリに移動して行う

docker compose up -d

動作しているか確認

$ docker compose ps
NAME                COMMAND                  SERVICE             STATUS              PORTS
launch-dev          "docker-entrypoint.s…"   app                 running             0.0.0.0:8000->8000/tcp

終了する時

docker compose down

localhost:8000でホームページが表示されるのを確認する
